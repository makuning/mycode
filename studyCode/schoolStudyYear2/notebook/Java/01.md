### 2、运行Java程序

> **通过javac程序将.java文件编译为.class文件**
>
> windows中：进入cmd命令窗口操作
>
> **注意事项**
>
> 1. 必须跟上.java后缀
> 2. 编译成功则生成.class文件（字节码文件）
> 3. 编译失败则不生成.class文件，且显示错误信息
>
> 例子：
>
> * javac Hello.java
> * javac HelloWorld.java

> **通过java程序执行.class文件**
>
> **注意事项**
>
> 1. 后跟类名，不能加.class后缀
>
> 例子：
>
> * java Hello
> * java HelloWorld
