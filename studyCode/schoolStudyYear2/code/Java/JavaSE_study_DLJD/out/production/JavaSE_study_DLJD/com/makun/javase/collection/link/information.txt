// 单链表数据结构：基本的单元是节点Node（可以自己写一写单链表）
    自写思路；
        一个Node类，表示节点
            节点中有两个属性
                1、存储的数据：Object
                2、下一节点的内存地址：Node
                3、有参构造方法

        一个Link类，表示单链表集合
                1、头节点Node
                2、添加方法
                3、删除方法
                4、修改方法
                5、查找方法
                6、size属性
                7、size() 方法
                8、lastNode() 递归方法（找到单链表中最后一个节点）
