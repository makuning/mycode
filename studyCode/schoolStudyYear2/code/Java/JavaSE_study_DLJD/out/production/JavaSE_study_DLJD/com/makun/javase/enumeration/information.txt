枚举类型

在开发中，可能遇到一个方法的执行结果，可能出现多种情况，三种，四种或者更多种，
并且是可以一枚一枚的列举出来的，这个时候boolean类型已经无法满足我们了，
而且返回数字会不规范，这时候就可以使用枚举类型
只有可以一枚一枚列举出来的才建议使用枚举类型
例如： 四季  星期  颜色  月份

枚举也是一种引用数据类型，编译后也是生成class文件
枚举中的每一个值可以看做时常量


// 枚举类型定义：
// 因为枚举值看作常量，所以建议枚举值大写（规范）
    enum 枚举类型名 {
        枚举值1,枚举值2,枚举值3
    }

// 给Result枚举类型定义了两个值
// SUCCESS表示成功
// FAIL表示失败
    enum Result {
        SUCCESS,FAIL
    }

// 枚举类型的使用：
    枚举类型.枚举类型中的值
    即：Result.SUCCESS

// 高版本的JDK的switch支持int、String、枚举
