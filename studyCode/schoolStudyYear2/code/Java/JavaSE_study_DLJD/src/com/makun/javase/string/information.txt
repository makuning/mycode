字符串类（final类，不能被继承）
// 普通字符串（不可扩展）
    String
    每次""双引号，就会在方法区的字符串静态池中创建一个字符串对象
    根据字符串创建方式的不同，变量可以直接去指向字符串静态池的对象
    或者是在堆中new的字符串对象去指向字符串静态池的字符串对象，然后这个引用去指向这个在堆中的字符串对象，
    在堆中的字符串对象又去指向方法区中的字符串对象

    这类字符串的值不能更改的意思并不是这个字符串引用装的值不能改变，只是在字符串静态池中的字符串是固定值的，
    它在类加载的时候会创建这个字符串，这个字符串是由一个char数组组成的，并且这个char数组是final的，所以不能更改
    但是在StringBuffer和StringBuilder中，这个char数组是没有final的，所以可以更改，而且扩展是使用了字符串的拷贝方法，
    将原来的字符串用一个更大的字符数组装起来

    在数字静态池中有这些【-128~127】数字，他们代码执行中不需要新建，这个是直接指向静态池中的，所以有时候在使用包装类用==比较时，
    如果定义时，值是这个范围内的数字，会返回true，因为他们指向的是同一个地址

    静态池为了缓存，提高程序的性能


// 可扩展字符串（线程具有安全性）
    StringBuffer
    构造时默认创建16长度的字符串
    append方法（加值）


// 可扩展字符串（线程具有不安全性）
    StringBuilder
    构造时默认创建16长度的字符串
    append方法（加值）
